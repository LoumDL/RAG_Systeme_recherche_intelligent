
Installez moodle-dl 
    pip install moodle-dl

Configurez-le pour votre Moodle :
    moodle-dl --init

Pour télécharger uniquement les PDF de tous vos cours
    moodle-dl --types pdf

Pour automatiser le telechargement 
    moodle-dl 

        pour le filtrage du telechargement
            moodle-dl -c
